Description

This is a project to try to make VR Haptic Glove technology more available for the public.

Currently, most VR Haptic Glove products out there are extremely expensive and are targeted for commercial use.

In starting this project, I wanted to create a budget, but still well-functioning VR haptic and finger tracking glove that's cheap and easy for people to get their hands on, and that works inside actual VR games.

What you'll need :

- 3D Print Files and Firmware - Driver - CH340 Driver - ESP32 Arduino IDE Install
Only 30$ per hand !
